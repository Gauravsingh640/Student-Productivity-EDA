# Student Productivity Analysis using EDA

## Overview

This project performs Exploratory Data Analysis (EDA) on the Student Productivity Dataset to identify the key factors affecting students' productivity and academic performance.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Key Insights

* Performed data cleaning and preprocessing.
* Analyzed relationships between academic and lifestyle factors.
* Created multiple visualizations to extract meaningful insights.
* Provided business recommendations based on the analysis.
